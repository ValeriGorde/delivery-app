# DeliveryApp

DeliveryApp - это ASP.NER Core (MVC) приложение для приемки заказа на доставку.

## Технологии

- Asp.Net Core Mvc
- EF / Entity Framework
- Code First
- C#
- SQLite
- AutoMapper

## Запуск проекта

 ```shell
   git clone https://github.com/yourusername/delivery-app.git
   cd delivery-app
   dotnet restore
   dotnet ef database update
   dotnet run
